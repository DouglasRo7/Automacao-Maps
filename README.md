# Automacao-Maps
Automação de pesquisa de estabelecimentos por cidade.

Antes de utilizar, deve colocar sua chave de API na variável "API_KEY" no arquivo "config.py".

Isso feito, só rodar o arquivo Python "Menu.py", que irá aparecer uma caixa de diálogo para difitar estabelecimento que quer encontrar + Cidade.

OBS: O programa foi parametrizado para retornar 5 estabelecimentos por pesquisa e disponibiliza-los em arquivos .xlsx e .json na mesma pasta que se encontra os arquivos da automação.
